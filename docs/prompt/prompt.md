# 최종 투자 리포트 목표주가 분석 프롬프트

최종 투자 리포트를 생성할 때 **증권사 목표주가와 자체 적정주가 분석 섹션**을 추가해줘.

해당 섹션에서는 단순히 목표주가 숫자만 제시하지 말고, **증권사 목표주가 범위, 자체 적정주가, 시나리오별 판단, 가격대별 대응 전략**까지 포함해서 분석해줘.

---

## 1. 최근 증권사 목표주가 확인

최근 **3~6개월 이내** 발행된 증권사 리포트를 기준으로 목표주가를 확인해줘.

가능한 경우 아래 항목을 표로 정리해줘.

| 증권사     |    리포트 발행일 | 투자의견     | 목표주가 | 주요 근거                    |
| ------- | ---------: | -------- | ---: | ------------------------ |
| 예: ○○증권 | YYYY.MM.DD | 매수/중립/매도 |  ○○원 | 실적 개선, 수주 증가, 밸류에이션 조정 등 |

확인 시 다음 기준을 지켜줘.

* 최근 리포트를 우선적으로 반영해줘.
* 오래된 리포트이거나 현재 실적·주가 상황과 맞지 않는 경우에는 **참고용**으로 표시해줘.
* 출처가 불분명하거나 확인이 어려운 목표주가는 사용하지 말고 **확실하지 않음**으로 표시해줘.
* 증권사별 목표주가 차이가 크다면, 왜 차이가 나는지 간단히 설명해줘.

---

## 2. 증권사 목표주가 범위 요약

증권사 목표주가를 종합해서 아래 형식으로 요약해줘.

> 최근 증권사 목표가는 대체로 **○만~○만 원 근처에 있습니다.**

가능하면 아래 항목도 함께 정리해줘.

| 구분       |  금액 |
| -------- | --: |
| 최저 목표주가  | ○○원 |
| 최고 목표주가  | ○○원 |
| 평균 목표주가  | ○○원 |
| 중간값 목표주가 | ○○원 |

단, 극단적으로 높거나 낮은 목표주가가 있다면 해당 값을 단순 평균에 포함하기보다, 왜 차이가 나는지 별도로 설명해줘.

---

## 3. 자체 적정주가 산정

증권사 목표주가와 별도로, 다음 요소를 반영해서 자체 적정주가를 산정해줘.

* 최근 실적
* 향후 실적 전망
* 매출 성장률
* 영업이익률
* 순이익률
* PER
* PBR
* ROE
* EV/EBITDA
* 업종 평균 밸류에이션
* 과거 평균 밸류에이션
* 성장성
* 재무 안정성
* 주요 리스크
* 최근 공시 및 뉴스 흐름

자체 적정주가는 하나의 숫자로 단정하지 말고, **보수적 / 기준 / 낙관 시나리오**로 나눠서 제시해줘.

---

## 4. 자체 적정주가 요약

아래 형식으로 자체 적정주가를 정리해줘.

| 구분       | 적정주가 범위 | 판단                   |
| -------- | ------: | -------------------- |
| 보수적 적정주가 | ○만~○만 원 | 실적 둔화 또는 리스크 반영      |
| 기준 적정주가  | ○만~○만 원 | 현재 실적 전망과 시장 기대 반영   |
| 낙관 적정주가  | ○만~○만 원 | 실적 서프라이즈와 성장 프리미엄 반영 |

각 적정주가를 산정할 때 적용한 핵심 가정을 간단히 설명해줘.

예시:

* 보수적 시나리오: 실적 성장 둔화, 마진 하락, 업종 멀티플 하락 반영
* 기준 시나리오: 현재 컨센서스와 최근 실적 흐름 유지
* 낙관 시나리오: 실적 서프라이즈, 수주 확대, 마진 개선, 성장 모멘텀 강화 반영

---

## 5. 가격대별 판단 표

현재 주가를 기준으로 가격대별 투자 판단을 표로 정리해줘.

| 주가 구간   | 판단     | 해석               | 대응 전략            |
| ------- | ------ | ---------------- | ---------------- |
| ○만 원 이하 | 저평가 구간 | 실적 대비 저평가 가능성    | 분할매수 고려          |
| ○만~○만 원 | 적정가 하단 | 기준 시나리오 기준 매력 존재 | 소액 또는 분할매수 가능    |
| ○만~○만 원 | 적정가 구간 | 기대와 실적이 어느 정도 균형 | 보유 또는 관망         |
| ○만~○만 원 | 고평가 구간 | 성장 기대가 상당히 반영됨   | 신규 매수 신중         |
| ○만 원 이상 | 과열 구간  | 낙관 시나리오 선반영 가능성  | 비중 축소 또는 차익실현 검토 |

가격대별 판단은 단순히 “매수/매도”로 단정하지 말고, 현재 실적과 기대감이 어느 정도 반영되어 있는지 중심으로 설명해줘.

---

## 6. 시나리오별 적정주가 판단

최종 투자 리포트에는 적정주가를 하나의 숫자로 단정하지 말고, **보수적 / 기준 / 낙관 시나리오**로 나눠서 제시해줘.

각 시나리오에는 다음 내용을 포함해줘.

| 시나리오     | 핵심 가정                                  |              적용 밸류에이션 | 예상 적정주가 | 현재 주가 대비 판단 |
| -------- | -------------------------------------- | --------------------: | ------: | ----------- |
| 보수적 시나리오 | 실적 성장 둔화, 마진 하락, 업종 멀티플 하락, 주요 리스크 현실화 |             낮은 멀티플 적용 | ○만~○만 원 | 고평가/주의      |
| 기준 시나리오  | 현재 실적 전망과 시장 컨센서스가 대체로 유지되는 경우         | 업종 평균 또는 최근 평균 멀티플 적용 | ○만~○만 원 | 적정/저평가      |
| 낙관 시나리오  | 실적 서프라이즈, 수주 확대, 마진 개선, 성장 모멘텀 강화      |           프리미엄 멀티플 적용 | ○만~○만 원 | 상승 여력 있음    |

각 시나리오별로 단순히 가격만 제시하지 말고, **그 시나리오가 성립하려면 어떤 조건이 충족되어야 하는지** 설명해줘.

또한 최종 판단에서는 아래 질문에 답해줘.

* 현재 주가는 어느 시나리오를 반영하고 있는가?
* 현재 가격에서 매수하면 어떤 시나리오에 베팅하는 것인가?
* 보수적 시나리오가 현실화될 경우 예상 하락 여력은 어느 정도인가?
* 낙관 시나리오가 현실화될 경우 예상 상승 여력은 어느 정도인가?
* 기준 시나리오 기준으로 현재 주가는 저평가 / 적정 / 고평가 중 어디에 가까운가?

아래 표도 함께 포함해줘.

| 현재 주가 위치      | 해석                     |
| ------------- | ---------------------- |
| 보수적 적정가 아래    | 실적이 크게 나쁘지 않다면 저평가 가능성 |
| 보수적~기준 적정가 사이 | 리스크는 있으나 매수 매력 존재      |
| 기준 적정가 근처     | 현재 기대와 실적이 대체로 균형      |
| 기준~낙관 적정가 사이  | 성장 기대가 상당 부분 반영        |
| 낙관 적정가 근처     | 좋은 뉴스가 이미 상당히 반영       |
| 낙관 적정가 초과     | 실적 확인 전 과열 가능성         |

---

## 7. 증권사 목표주가와 자체 적정주가 비교

증권사 목표주가와 자체 적정주가를 비교해서 아래 형식으로 정리해줘.

| 구분          |   주가 범위 | 해석          |
| ----------- | ------: | ----------- |
| 현재 주가       |     ○○원 | 현재 시장 가격    |
| 증권사 목표주가 범위 | ○만~○만 원 | 시장 컨센서스 기준  |
| 자체 보수적 적정주가 | ○만~○만 원 | 리스크 반영 기준   |
| 자체 기준 적정주가  | ○만~○만 원 | 현실적 기준 시나리오 |
| 자체 낙관 적정주가  | ○만~○만 원 | 성장 기대 반영 기준 |

비교 후 다음 내용을 설명해줘.

* 현재 주가가 증권사 목표주가 대비 저평가인지 고평가인지
* 현재 주가가 자체 기준 적정주가 대비 저평가인지 고평가인지
* 증권사 목표주가와 자체 적정주가가 차이 나는 이유
* 현재 주가가 어떤 기대를 선반영하고 있는지

---

## 8. 최종 판단

마지막에는 아래 형식으로 요약해줘.

| 항목          | 내용                        |
| ----------- | ------------------------- |
| 현재 주가 기준 판단 | 저평가 / 적정 / 고평가 / 판단 유보    |
| 기준 적정주가     | ○만~○만 원                   |
| 증권사 목표주가 범위 | ○만~○만 원                   |
| 보수적 적정주가    | ○만~○만 원                   |
| 낙관 적정주가     | ○만~○만 원                   |
| 1차 관심 매수가   | ○만 원 이하                   |
| 추가 매수 검토 구간 | ○만~○만 원                   |
| 주의 구간       | ○만 원 이상                   |
| 핵심 확인 포인트   | 실적, 수주, 마진, 밸류에이션, 리스크 요인 |

최종 코멘트는 아래 형식으로 작성해줘.

> 현재 주가 기준으로 이 종목은 **저평가 / 적정 / 고평가** 구간에 가깝습니다.
> 기준 시나리오상 적정주가는 **○만~○만 원**으로 판단되며, 현재 주가는 **○○ 시나리오**를 어느 정도 반영하고 있습니다.
> 따라서 신규 매수는 **○○ 구간 이하에서 분할매수**, 보유자는 **실적과 핵심 지표를 확인하며 대응**하는 전략이 적절합니다.

---

## 9. 주의 문구

목표주가는 확정값이 아니라 추정치임을 명시해줘.

다음 문구를 반드시 포함해줘.

> 위 적정주가는 현재 확인 가능한 실적, 컨센서스, 밸류에이션, 업종 상황을 바탕으로 한 추정입니다.
> 향후 실적 발표, 공시, 수주, 금리, 환율, 시장 심리, 업종 멀티플 변화에 따라 적정주가는 달라질 수 있습니다.
> 따라서 목표주가는 매수·매도 확정 신호가 아니라 투자 판단을 돕기 위한 참고 기준으로 봐야 합니다.

---

## 10. 작성 방식

리포트 작성 시 다음 원칙을 지켜줘.

* 단정적인 표현은 피하고, 근거를 함께 제시해줘.
* 추정이 필요한 경우 **추정입니다**라고 표시해줘.
* 출처가 불분명한 정보는 **확실하지 않음**으로 표시해줘.
* 실적, 밸류에이션, 증권사 리포트, 뉴스, 공시를 종합해서 판단해줘.
* 단순히 목표주가만 제시하지 말고, 현재 주가가 어떤 기대를 반영하고 있는지 설명해줘.
* 투자 판단은 **매수/매도 추천**이 아니라, 가격대별 대응 전략 중심으로 작성해줘.
