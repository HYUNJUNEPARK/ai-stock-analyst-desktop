**반복 등장하는 핵심 이슈**

- 분기 실적 (1Q 2026: 순익 1.6조원, 역대 최대, YoY +9.0%)
- 주주환원 정책: 밸류업 2.0 (ROE·성장률 연동 상한 없는 주주환원율, DPS 매년 10% 이상 확대, 2026~2028)
- 글로벌 IR: CEO 진옥동 직접 북중미(미·멕·캐) 기관투자자 로드쇼 (2026-05-10~22)
- 서울시 금고: 신한은행, 2027~2030년 주·부금고 동시 수성 (연 51조원 규모)
- 포용금융 리스크: SEC 보고서에 정부 주도 포용금융이 연체율·대손비용 악화 가능성 공식 기재
- 스트레스 DSR 3단계: 가계대출 증가율 1.5% 이내 총량 규제 → 이자이익 성장 제약
- 자회사 CEO 교체: 신한라이프·신한자산운용 수장 교체 (2026-05-05), 조직 긴장감 목적
