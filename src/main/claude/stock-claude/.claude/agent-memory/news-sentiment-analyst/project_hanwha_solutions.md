3. **IRA 정책 리스크**: 트럼프 빅뷰티빌(One Big Beautiful Bill) 내 주거용 ITC 조기 일몰, AMPC 폐지 시도 존재. 공화당 일부 의원 반대로 현재까지 AMPC 존치되나 지속 모니터링 필요.
4. **케미칼 부문 2년 반 만의 흑자전환**: 1Q26 영업익 341억원. PVC 해외 사업 흑자, W&C 고수익 제품 비중 확대.
5. **카터스빌 셀 공장 가동 지연**: 당초 2025년 4Q 양산 예정이었으나 유틸리티 장비 결함으로 2026년 3Q로 연기. 웨이퍼는 2026년 2월부터 양산 개시.
6. **재무구조 압박**: 총차입금 약 12.5조원, 금감원이 약 5조원 규모 비업무용 자산(한화임팩트 47.9%, 한화호텔앤드리조트 49.57% 지분 등) 매각 미검토를 문제로 지적.
