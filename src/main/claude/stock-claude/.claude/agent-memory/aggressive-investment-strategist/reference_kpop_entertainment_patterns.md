## K-POP 엔터 업종 공통 리스크 분류

### 1. 아티스트 집중 리스크
- 단일 아티스트 매출 의존도가 50% 초과 시 이탈·공백 리스크를 별도 시나리오로 모델링
- 측정 방법: 해당 아티스트 투어·앨범 제거 시 영업이익률 변화 추정
- 하이브의 경우 BTS가 동시에 모멘텀 집중 원천이자 이 리스크의 주체 (이중성)

### 2. 멀티레이블 고정비 구조
- 여러 레이블 운영 시 OPM은 단레이블사 대비 낮지만, 레버리지 효과 잠재력 보유
- 투자 판단 시 절대 OPM이 아닌 레버리지 전환 시점(BEP 매출 달성 여부)으로 판단
- JYP(OPM 19.5%)와 직접 비교는 사업 모델 차이 무시, 사과-사과 비교 아님

### 3. 소송·이미지 리스크 계량화
- 소송가액 / 시총 비율로 금전적 임팩트 산정 (하이브 431억 / 9.81조 = 0.44%, 중요도 낮음)
- 별도 정성 평가: 소송이 팬덤 이탈 → 아티스트 계약 협상력 저하로 전이되는지 여부
- 기준: 미디어 재확산 빈도와 관련 아티스트 음원/앨범 성적 변동 연계 분석

### 4. 앨범 판매 성숙화 vs. 플랫폼 전환
- 국내 앨범 판매 YoY -9%, 디지털 소비 -6.4%는 구조적 성숙화 신호
- 플랫폼(Weverse 등) MAU·유료전환율 및 콘서트 매출 비중 증가로 상쇄 가능한지 확인
- 앨범 판매 감소폭이 -15% 이상으로 악화 시 업종 긍정 전망 재검토 트리거

### 5. 투어 수익 인식 사이클
- 엔터 종목의 이익 가시성은 투어 일정과 수익 인식 시점에 연동
- 투어 발표 → 실적 인식 리드타임: 통상 6~12개월
- 투어 종료 후 공백기(모멘텀 단절) 리스크는 멀티IP 가동 여부로 헤지

## 밸류에이션 기준값 (2026년 K-POP 빅4 기준)
- 업종 평균 PER: 25~40배
- 성장 프리미엄 허용 PER: 최대 45~50배 (구조적 성장 업황 + 플랫폼 옵션 가치 반영)
- PER 60배 이상: 이익 성장 가속 확인 전까지 분할 진입 원칙 적용
- PBR 기준: 업종 평균 2~4배 (중상위권 기준)

## 한한령 해제 모니터링 기준
- 단계적 해제: 중국 K-POP 이벤트 시장 2033년 221.7억달러(CAGR 10.3%) 전망
- 모니터링 지표: 중국 내 K-POP 콘서트 개최 건수, 중국 플랫폼 내 스트리밍 수치
- 미중 갈등 재고조 시 해제 속도 지연 → 업종 성장 시나리오 가정치 하향 조정 필요
