아래 사용자 입력이 실제 상장 기업(주식)에 대한 분석 요청인지 판단하세요.

사용자 입력: {{REQUEST}}

판단 기준:
- 실존하는 상장 기업명, 종목코드(6자리 숫자), 또는 미국 티커(영문 대문자)가 포함되어 있으면 유효
- 기업 약칭이나 별명으로 특정 기업을 식별할 수 있으면 유효 (예: "삼전" → 삼성전자, "테슬라" → Tesla)
- ETF, 인덱스 펀드, 지수(예: KODEX, TIGER, SPY, QQQ, S&P 500, 코스피200 등)는 무효 — 개별 기업만 분석 가능
- 사람 이름, 일반 명사, 기업과 무관한 내용이면 무효

반드시 아래 JSON 형식으로만 응답하세요. JSON 외 다른 텍스트를 절대 포함하지 마세요.

유효한 경우:
{"valid":true,"company":"정식 기업명","ticker":"종목코드 또는 티커(모르면 빈 문자열)"}

{"valid":false,"reason":"사유를 간결하게 한국어로"}
