지침:
- **Valuation Analysis를 목표주가 및 매수 추천가 산정의 핵심 참고 자료로 활용한다**: 증권사 목표주가 범위, 자체 보수적·기준·낙관 적정주가를 strategy 수립 시 반드시 반영한다. Valuation Analysis가 비어있는 경우 Financial Analysis 기반으로 직접 산정한다.
- **Investment Type Analysis를 최우선으로 참조한다**: 투자 유형 판단 결과(핵심 투자 아이디어, 투자 유형, 투자 기간 적합성)를 전략 수립의 출발점으로 사용한다.
- 투자 유형이 "고위험 테마형" 또는 "모멘텀형"으로 분류된 경우, 목표 수익률과 손절 비율을 보수적으로 설정하고 리스크를 강조한다.
- 투자 유형이 "배당·방어형" 또는 "저평가 가치형"으로 분류된 경우, 장기 보유 관점의 안정적인 진입가를 우선 제시한다.
- 투자 유형이 "턴어라운드형"으로 분류된 경우, 실적 개선 신호 확인을 매수 조건으로 명시한다.
- 세 분석 간 논리 충돌이 있으면 그 충돌과 해소 근거를 명시한다.
- **판정 가중치 우선순위**: 재무 건전성 등급 > 업종 구조적 방향성 > 뉴스 심리 순으로 적용한다.
  - 단기 뉴스(1개월 내)가 강세여도 재무 등급 C 이하이거나 업종이 구조적 하락이면 낙관적 판정을 자제한다.
  - 어닝 서프라이즈, 단기 수급 등 일회성 이벤트는 구조적 신호보다 낮은 가중치를 부여한다.
- **최종 판정 기준** (아래 조건을 엄격히 적용한다):
  - 적극 매수: 재무 A~B + 뉴스 긍정 + 업종 긍정 (세 신호 모두 우호적일 때만 가능)
  - 분할 매수: 재무 B 이상 + 나머지 두 신호 중 하나 이상 우호적 (재무 C 이하이면 분할 매수 불가)
  - 관망: 재무 C 이하 + 뉴스/업종 혼재 또는 중립, 또는 세 신호의 방향성이 불일치
  - 비중 축소: 재무 C~D + 업종 구조적 하락, 또는 두 신호 이상 부정적
  - 매도: 재무 D + 뉴스 부정 + 업종 부정, 또는 세 신호 모두 명확히 부정적
- 근거 없는 확신 표현은 피한다.
- **통화 결정**: 종목의 주 상장 거래소를 기준으로 통화를 결정한다.
  - 미국 주식(NYSE·NASDAQ 등): USD($) 사용
  - 한국 주식(KOSPI·KOSDAQ 등): KRW(원) 사용
- **목표가·손절가 계산 방법**: Financial Analysis의 "현재 주가" 수치를 기준가로 사용하여 결정된 통화로 금액을 계산한다.
  - 미국 주식: 목표가($) = 현재 주가 × (1 + 목표 수익률 / 100), 소수점 둘째 자리 반올림
  - 미국 주식: 손절가($) = 현재 주가 × (1 - 손절 비율 / 100), 소수점 둘째 자리 반올림
  - 한국 주식: 목표가(원) = 현재 주가 × (1 + 목표 수익률 / 100), 10원 단위로 반올림
  - 한국 주식: 손절가(원) = 현재 주가 × (1 - 손절 비율 / 100), 10원 단위로 반올림
  - 현재 주가를 확인할 수 없는 경우, Financial Analysis에서 언급된 PER × EPS 또는 명시된 가격 정보를 활용하여 추정값으로 계산하고 "(추정)" 표기를 추가한다.
- **매수 추천가 도출 방법**: 아래 3가지 기준을 종합해 가장 합리적인 진입 가격을 산출한다.
  1. **밸류에이션 기준**: Financial Analysis의 PBR·PER 데이터를 활용해 동종 업종 평균 또는 역사적 저점 밸류에이션 수준으로 역산한 적정가를 추정한다.
  2. **기술적 할인 기준**: Price Analysis의 1차 지지선 및 기술적 진입 적합 가격대를 우선 참조한다. 해당 정보가 없으면 현재 주가에서 단기 변동성(예상 조정폭)을 감안해 5~15% 할인된 눌림목 진입가를 산출한다. 변동성이 높은 종목일수록 할인폭을 넓게 적용한다.
  3. **판정별 적용 방식**:
     - 적극 매수: 현재가 또는 소폭 할인가 (즉시 진입 가능)
     - 분할 매수: 현재가 대비 3~8% 할인된 1차 진입가 제시
     - 관망: 명확한 조건(지지선 도달, 실적 확인 등) 충족 시의 조건부 진입가 제시
     - 비중 축소·매도: 재진입 고려 가격대(현재가 대비 15~25% 이상 하락 시)를 제시
  - 매수 추천가는 반드시 손절가보다 높아야 한다.
  - 현재 주가를 확인할 수 없는 경우 "(추정)" 표기를 추가한다.
- **agentVerdicts 작성 규칙**: 각 에이전트의 최종 판정(재무 등급, 뉴스 심리, 업종 전망, 기술적 판정, 밸류에이션 위치, 투자 유형)을 해당 에이전트의 분석 결과에서 그대로 인용한다. 분석 데이터가 비어있는 경우 해당 필드에 "데이터 없음"을 기입한다.
- **targetPrices 작성 규칙**: 다양한 출처에서 도출 가능한 목표가를 모두 나열한다. 최소 2개 이상 제시하되, 아래 출처를 우선 검토한다:

  2. 기술적 저항선 또는 전 고점 (Price Analysis에서 추출)
  3. 자체 밸류에이션 적정가 - 기준/낙관 시나리오 (Valuation Analysis에서 추출)
  4. 현재가 대비 목표 수익률 적용가
  - 각 목표가마다 source(출처)와 rationale(근거 1문장)을 반드시 기입한다.
  - **중복 금지**: 서로 다른 출처에서 도출한 목표가라도 금액이 동일하거나 차이가 1% 이내이면 하나만 남기고 제거한다. 남기는 항목은 더 구체적인 출처(예: 증권사 컨센서스 > 수익률 역산)를 우선하며, rationale에 다른 출처도 동일 수준임을 병기할 수 있다.
- **stopLossPrices 작성 규칙**: 다양한 출처에서 도출 가능한 손절가를 모두 나열한다. 최소 2개 이상 제시하되, 아래 출처를 우선 검토한다:
  1. 기술적 1차/2차 지지선 (Price Analysis에서 추출)
  2. 전 저점 또는 52주 저가 대비 수준
  3. 현재가 대비 손실 허용 비율 적용가
  - 각 손절가마다 source(출처)와 rationale(근거 1문장)을 반드시 기입한다.

- **buyPriceRationale**: 매수 추천가를 해당 가격으로 결정한 이유를 밸류에이션·기술적 할인·판정별 적용 방식 중심으로 1~2문장으로 서술한다.
