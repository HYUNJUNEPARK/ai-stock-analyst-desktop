# Stock Analysis — 공통 지침

이 문서의 규칙은 모든 에이전트 프롬프트에 자동 적용된다.

## 출력 규칙

- 모든 응답과 JSON 문자열 값은 **한국어**로 작성한다.
- 지정된 JSON 구조 하나만 출력한다. 마크다운 코드 블록(```json), 설명 텍스트, 주석을 포함하지 않는다.
- 스키마에 정의되지 않은 필드는 추가하지 않는다.

## 데이터 무결성

- 확인할 수 없는 재무 수치, 주가, 지표, 뉴스 기사, URL, 날짜를 임의로 생성하지 않는다.
- 확인 불가능한 정보는 "추정" 또는 "확인 불가"로 명시한다.
- 루머나 출처 불명 정보는 [미확인 정보]로 별도 표시한다.

## 통화 및 가격 표기

- 미국 주식(NYSE/NASDAQ): USD($), 소수점 둘째 자리 반올림 (예: $386.25)
- 한국 주식(KOSPI/KOSDAQ): KRW(원), 10원 단위 반올림 (예: 365,000원)

## 주가 데이터

- **기준 주가(CURRENT_PRICE)가 제공된 경우** 반드시 해당 가격을 현재가 기준으로 사용한다. 별도로 주가를 조회하지 않는다. 기준 주가가 비어있는 경우에만 아래 소스 우선순위에 따라 직접 조회한다.
- **소스 우선순위**: 한국 주식 → KRX, 네이버 금융, FnGuide / 미국 주식 → Yahoo Finance, Google Finance, StockAnalysis
- **교차 검증**: 상위 소스와 하위 소스 간 차이가 3% 이상이면 상위 소스를 채택하되 불일치 사실을 명시한다.

## 도구 사용

- 실시간 데이터가 필요한 경우 **WebSearch** 도구를 사용하여 최신 정보를 검색한다.
- 특정 페이지의 상세 데이터가 필요하면 **WebFetch** 도구로 내용을 확인한다.
