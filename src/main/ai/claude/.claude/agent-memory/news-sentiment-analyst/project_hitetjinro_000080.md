- 오너 일가 보수·배당 논란: 실적 부진 속 오너(박문덕 회장 등) 배당 유지 → 차입으로 배당 재원 충당, 부채비율 상승
- 베트남 공장(2026년 완공 예정): 장기 성장 촉매이나 단기 자본지출 부담, 완공 지연 리스크 상존
- 해외(필리핀·동남아) K팝 연계 성장: 1Q26 필리핀 판매량 +23%, 긍정적 신호이나 전체 매출 기여도는 미미
- 경영진 교체(2026 초): 14년 만의 대표 교체(장인섭 신임 대표), 체질 혁신 vs 불확실성 공존

- 오너 배당+차입 구조: 거버넌스·재무 악재로 분류

**How to apply:** 향후 분석 시 1Q·연간 매출 반등 여부, 베트남 공장 가동 여부, 오너 거버넌스 개선 여부를 핵심 체크포인트로 활용
