3. **IRA 정책 리스크**: 트럼프 빅뷰티빌(One Big Beautiful Bill) 내 주거용 ITC 조기 일몰, AMPC 폐지 시도. 공화당 일부 의원 반대로 현재까지 AMPC 존치되나 지속 모니터링 필요.

6. **재무구조 압박**: 총차입금 약 12.5조원, 신용등급 AA-(부정적).
7. **非FEOC 프리미엄 수혜**: 2026년부터 FEOC 규제 본격화, 중국산 제품 미국 시장 퇴출로 한화 직접 수혜.
