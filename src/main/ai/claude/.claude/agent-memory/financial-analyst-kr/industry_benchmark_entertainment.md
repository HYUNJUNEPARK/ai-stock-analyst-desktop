## 업종 특성
- K팝 엔터테인먼트는 아티스트 활동 사이클에 따른 실적 변동성이 매우 큼
- 음반, 공연, 콘텐츠(플랫폼), 굿즈(MD) 등 다각화된 수익 구조
- 고정비(연습생 육성, 제작비) 부담 크고, 신인 데뷔 시 초기 비용 집중
- 저부채 구조가 일반적 (제조업 대비 설비투자 적음)

## 빅4 업종 평균 지표 (2023~2024 기준)
| 지표 | 빅4 평균 추정 | 비고 |
|------|-------------|------|
| 영업이익률 | 10~15% | JYP 20%+로 최고, 하이브 8~14% |
| 순이익률 | 5~12% | |
| ROE | 10~20% | JYP 25%+ 최고 |
| 부채비율 | 50~100% | 전반적으로 저부채 |
| 유동비율 | 150~250% | 현금 풍부 구조 |
| PER | 20~60배 | 성장주 프리미엄, 변동 큼 |
| PBR | 2~5배 | |

## 기업별 특성
- 하이브: 규모 1위(매출 2조+), 멀티레이블, BTS 의존도 높음, 플랫폼(위버스) 투자
- JYP: 수익성 1위(영업이익률 20%+), 효율적 운영, 트와이스·스트레이키즈·뉴진스 등 IP 다양
- SM: 에스파·NCT 등 IP 강점, 카카오 인수 후 거버넌스 안정화
- YG: 블랙핑크 의존도, 상대적으로 저평가 구간
