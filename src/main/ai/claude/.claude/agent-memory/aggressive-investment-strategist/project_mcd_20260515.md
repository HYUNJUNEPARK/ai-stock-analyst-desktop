맥도날드(MCD) 분석 결과 (2026-05-15): 세 분석가 모두 '중립' 수렴 → 분할 매수 선택.

**핵심 판단 구조:**
- 재무 분석가: B+ (영업이익률 45~47%, FCF 70억 달러, ROIC 21% — 업종 최상급 수익성)
- 뉴스 감성 분석가: 중립 (Q1 실적 서프라이즈 vs. CEO의 미국 SSS 마이너스 전환 직접 인정)
- 업종 리서처: 중립 (QSR 사이클 저점, SSS 회복이 재평가 촉매)

**최종 판정:** 분할 매수 (1차 50% 즉시, 2차 50%는 Q2 SSS 플러스 전환 확인 후)

**목표가:** 365달러 (+27%, UBS 기준) / 공격적 시나리오 385달러 (+35%)
**손절:** -10% (SSS 3분기 연속 악화 시 조기 청산)
**R/R:** 2.7:1 (기본) / 3.5:1 (공격)
**포지션:** 포트폴리오 8~10%
**투자 기간:** 중기 12~18개월

**신호 충돌 해소 프레임:**
- 세 분석가 모두 '중립' 수렴 = 트리거 명확한 압축 구간으로 해석
- CEO 소비 악화 경고는 단기 역풍이지 구조적 훼손 아님 (FCF 70억/이자보상배율 7.5~8배가 완충재)
- 관건: Q2 미국 SSS +1% 이상 반전 시 2차 매수 트리거 발동

**Why:** 세 분석가 신호가 모두 동일 변수(SSS 회복)를 지목할 때, 트리거 발동 전 선제 진입이 공격적 전략의 본질. 관망보다 1차 진입 후 확인 매수 방식이 업사이드 극대화에 유리.

**How to apply:** 유사 구조(압도적 수익성 + 단기 트래픽 둔화 + 명확한 회복 촉매 대기) 종목에 동일 프레임 적용 가능. [[feedback-signal-conflict-resolution]]
