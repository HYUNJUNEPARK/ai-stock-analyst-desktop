## QSR 성숙 브랜드 공통 패턴

### 구조적 강점 (분석 시 항상 확인)
- **프랜차이즈 헤비 모델**: 영업이익률 40~50%대 가능. 자산 경량화 + 가맹점주에게 리스크 이전
- **배당 귀족주 여부**: 연속 배당 증가 25년 이상이면 하방 방어선으로 활용. 경영진이 배당 삭감보다 자사주 매입 축소를 먼저 선택하는 경향
- **디지털 충성도 프로그램**: 앱 회원 수억 명 규모면 ARPU 제고 여지 존재. 단기 방문 감소를 상쇄하는 중기 레버

### 공통 리스크 패턴
1. **가격 인상 누적 반작용**: 2~3년간 누적 가격 인상 30~40% 이후에는 저소득층 이탈이 구조화될 수 있음. McValue류 가격 방어 전략이 나오면 이미 반작용이 시작된 신호
2. **최저임금 입법 연동 가맹점주 리스크**: 가맹점주 수익성 악화 → 신규 출점 의욕 감소 → 성장 목표 미달로 연결되는 경로
3. **지정학적 반감 리스크**: 미국 브랜드 특성상 반미 감정, 불매운동, 지역 분쟁이 동일매장매출에 직접 반영. CEO 언급 기준으로 수치화 가능
4. **유가 + 저소득층 방문 상관관계**: 유가 $4/갤런 이상 = 저소득층 외식 감소 선행 지표. 주유비 급등 시 QSR 방문 빈도 감소 빠르게 나타남
5. **레버리지 구조 + 고금리 리스크**: 자사주 매입으로 PBR 음수인 종목의 경우 Net Debt/EBITDA로 레버리지 측정. 2.5배 초과 + 금리 상승 구조 = 차환 비용 민감도 높음

### 동일매장매출(SSS) 해석 기준
- +3% 이상: 강세, 볼륨+가격 동반 성장
- +1~3%: 보통, 가격 인상분만 반영 가능성
- 0~+1%: 경고, 실질 방문객 감소 중
- 마이너스: 위험, 2분기 연속 시 구조적 이탈 가능성

### 분할 매수 트리거 조건 (맥도날드 사례 기반)
- 실적 발표일 주가 하락 (기대치 낮아진 신호)
- 배당 귀족주 지위 유지 중
- CEO 가이던스 하향으로 컨센서스가 이미 낮아진 상태
- 신흥 시장 출점 계획 유효
- 위 4가지 동시 충족 시 분할 매수 유효
