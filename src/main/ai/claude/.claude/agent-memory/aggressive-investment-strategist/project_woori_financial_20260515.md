## 핵심 판단 요약

**판정:** 분할 매수 (세 분석가 모두 중립 신호 → 공격적 비대칭 해석으로 분할 매수 전환)

**Why:** 세 분석가 모두 중립 신호를 냈으나 가격 비대칭(업사이드 30~48% vs. 손절 -12%)과 비과세 배당 재원 6.3조원이라는 5년 장기 주주환원 가시성이 공격적 포지션을 정당화. 단, 7월 임시주총 가처분 소송 이벤트를 분산시키기 위해 3분할 진입 구조를 채택.

**How to apply:** 금융지주 분석에서 세 분석가 모두 중립 신호를 보낼 때, 업사이드/다운사이드 비율이 2.4:1 이상이고 배당 안전망이 확인되면 분할 매수를 선택하는 기준으로 활용.

## 신호 충돌 해소 방식

| 분석가 | 신호 | 처리 방식 |
|--------|------|----------|
| 재무 분석가 | 중립~약한 매수 (B등급, PBR 0.65배) | 기본 축 — 저평가 확인으로 하방 지지 |
| 뉴스 감성 분석가 | 중립 (어닝 쇼크 vs. 배당 공약) | 어닝 쇼크를 '소화된 악재'로 재해석 |
| 업종 리서처 | 중립~관망 (비은행 열위) | 구조적 할인이 현재가에 반영됐다고 판단 |

세 분석가가 동시에 중립 수렴 + 업사이드 비율 2배 이상 → 분할 매수 채택 (이 패턴 반복 확인 필요)

## 우리금융 특이 리스크 요인 (타 금융지주와 차별되는 점)

1. **비은행 기여도 17%** — KB(37%), 신한(29%) 대비 구조적 취약, 리레이팅 속도 제한
2. **동양생명 소액주주 가처분 소송** — 7월 24일 임시주총 리스크 (주식교환 무효화 시 CET1 재하락)
3. **내부통제 반복 이슈** — 거버넌스 프리미엄 할인 상존, 타 금융지주 대비 할인 요인
4. **우리투자증권 자본 소요** — 종투사 달성에 8,000억원 이상 필요, 배당 재원 잠식 가능성

## 분할 매수 트리거 기준 (이 종목 전용)

- **1차 진입 (30%):** 즉시 (32,500원 수준에서 PBR 저평가 확인)
- **2차 진입 (40%):** 7월 24일 임시주총 가처분 소송 부결 확인 후
- **3차 진입 (30%):** 2Q26 실적 발표 후 수정 순이익 YoY +8% 복원 확인
- **손절 기준:** 28,600원 (-12%) / BPS 대비 58% 수준
