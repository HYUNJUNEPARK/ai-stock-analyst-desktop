## 글로벌 QSR 업종 공통 리스크 및 모니터링 패턴

### 핵심 리스크 유형

**1. 동일점포매출(SSS) 사이클 리스크**
- SSS 역성장 구간은 통상 2~4분기 지속 후 반전
- CEO의 공개 경고는 컨센서스 이미 반영 신호 — 추가 하락보다 바닥 확인 국면으로 해석
- SSS +1% 이상 반전 확인 시점이 가장 강력한 매수 트리거
- 선행 지표: 미국 소비자신뢰지수, CPI 식품 항목, 로열티 앱 활성 사용자 수

**2. 원가·마진 압박 리스크**
- 최저임금 상승 + 포장재 규제 강화 → 키오스크·AI 주문 자동화 투자 병행 필수
- 관세·공급망 충격은 1~2분기 내 마진 가이던스 하향으로 현실화
- 모니터링: 분기 영업이익률 변동 ±2%p 이상 발생 시 구조적 훼손 여부 재검토

**3. 프랜차이즈 모델 특성 리스크**
- 직영 대비 수익 안정성 높으나, 가맹점 재무 건전성 악화 시 폐점 → 로열티 수입 감소
- 리오프닝·확장 시장(중국, 신흥국) 출점 속도가 선진국 SSS 부진 상쇄 가능 여부 확인

**4. 밸류에이션 멀티플 수축 리스크**
- PER 26~28배: SSS 성장 재가속 시 30배 확장 / SSS 지속 역성장 시 22~23배 수축
- 멀티플 수축 -15% + SSS 역성장 -5~10% = 최대 -25% 하락 시나리오

**5. 브랜드·지정학 리스크**
- 중동 불매운동은 지역 SSS에 직접 영향, 재확산 여부 모니터링
- 소셜미디어 브랜드 이슈는 단기 노이즈 vs. 지속 불매로 구분 필요

### 프랜차이즈 고마진 구조에서의 공격적 전략 기준
- 영업이익률 40% 이상 + FCF 마진 20% 이상 + 이자보상배율 5배 이상 = 단기 역풍 소화 가능 구간
- 위 조건 충족 시: SSS 역성장 = 매수 기회로 해석, 사이클 저점 분할 진입 유효
