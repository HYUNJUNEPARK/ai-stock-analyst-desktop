description: 재무 건전성 경고(C등급, 부채비율 196%)와 단기 실적 개선(1Q26 영업이익 205% 급증)이 충돌할 때 분할 매수를 선택한 근거 및 프레임
type: feedback
---

## 규칙
재무 건전성 등급이 C 이하이더라도, 이하 3가지 조건이 동시 충족될 경우 즉시 매도/관망 대신 분할 매수를 선택한다:
1. 분기 실적이 전년 대비 100% 이상 급증하며 전 부문 흑자 전환
2. PBR이 1.0배 미만으로 순자산 대비 할인 상태
3. 구조적 성장 업종(CAGR 10% 이상)의 정책 수혜가 확인된 상태

## Why:
한화솔루션 사례에서 재무 C등급(부채비율 196%, 2년 연속 영업적자)과 1Q26 영업이익 205% 급증·전 부문 흑자 전환이 충돌. 재무 정량 데이터를 기반으로 하되, 전환점(inflection point) 진입 초기에 과도하게 보수적 판단을 내릴 경우 업사이드의 상당 부분을 포기하게 됨. PBR 0.82배 저평가는 최악 시나리오가 이미 반영됐다는 신호로 해석.

## How to apply:
신호 충돌이 발생할 때 단순히 숫자 카운팅(호재 n건 vs. 악재 m건)으로 판정하지 않는다. 대신 (1) 전환점 여부, (2) 밸류에이션 안전 마진, (3) 미결 이벤트 집중도를 종합해 판정 등급과 진입 방식(풀포지션 vs. 분할)을 결정한다. 미결 이벤트가 6주 내 집중되어 있을 경우, 이벤트 트리거 기반 분할 진입 일정을 설계한다.
