- 최종 판정: 분할 매수 (목표가 60,000원, 손절 38,000원, R/R 3.25:1)

## Why:
재무 건전성이 C→D로 악화되었음에도 분할 매수를 유지한 이유는 업종 구조적 수혜(비FEOC 해자, 45X AMPC, 미국 셀 공장 완성)가 더욱 구체화되었기 때문. 단, 선반영 부담과 D등급을 반영해 초기 비중을 축소하고 이벤트 트리거를 더 엄격하게 적용.

## How to apply:
재무 등급이 C→D로 악화되더라도 업종 구조적 수혜가 강화되는 방향이면 분할 매수 판정을 유지하되 초기 진입 비중을 1%p 축소하는 룰을 적용. 52주 대비 80% 이상 반등한 종목에서는 목표가 상단을 유지하되 R/R 기준을 재검토할 것.
